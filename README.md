# oh-my-termux

Setup Termux with:

 - Solarized Dark
 - Ubuntu Mono with powerline patch
 - oh-my-zsh

# Requirement
- curl
```bash
pkg update && pkg install
pkg install -y libcurl curl
```

# Installation
```bash
bash -c "$(curl -fsSL https://git.io/oh-my-termux)"
```
# Let's start
```bash
nano .zshrc
```
